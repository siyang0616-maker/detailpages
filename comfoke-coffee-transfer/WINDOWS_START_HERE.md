# Windows 작업 시작 메모

이 폴더는 컴포크커피 양도양수 상세페이지 초안입니다. 현재 구조는 빌드가 필요한 앱이 아니라 정적 HTML 페이지입니다.

## 바로 열기

- Windows: `open-preview.bat` 더블클릭
- 직접 열기: `index.html` 더블클릭
- 편집 시작: `index.html`, `README.md`

## 현재 포함된 작업물

- `index.html`: 12개 컷 구조의 상세페이지 초안
- `README.md`: 제작 기준과 실사용 전 확인할 자료 목록

## Windows에서 확인할 점

- HTML 문서 인코딩: UTF-8
- 별도 Node/npm 설치: 필요 없음
- 별도 빌드 명령: 필요 없음
- 실제 이미지 파일 없이 HTML/CSS로 먼저 구성한 초안

## 새 작업 전 체크리스트

- 실제 매장 사진, 매출 증빙, 권리금/보증금/월세/관리비 자료가 들어오면 README의 확인 필요 항목부터 갱신
- 브랜드명이 `컴포크커피`가 아니라 `컴포즈커피` 등 다른 표기라면 `index.html`, `README.md`, 이 파일의 브랜드명을 먼저 일괄 수정
- 사진 컷을 PNG로 따로 만들고 싶으면 `images/` 폴더를 추가하고 `index.html`에서 이미지형 구조로 바꾸면 됨

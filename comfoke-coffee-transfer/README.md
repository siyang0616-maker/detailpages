# 컴포크커피 양도양수 상세페이지

컴포크커피 양도양수 매물 상세페이지 초안입니다.

## 포함 파일

- `index.html`: 브라우저에서 바로 확인 가능한 상세페이지 초안
- `open-preview.bat`: Windows에서 미리보기 페이지를 바로 여는 실행 파일
- `WINDOWS_START_HERE.md`: Windows에서 이어 작업하기 위한 시작 메모

## Windows에서 이어 작업하기

1. `open-preview.bat`을 더블클릭해서 현재 상세페이지를 확인합니다.
2. 문구, 색상, 구성은 `index.html`에서 수정합니다.
3. 실제 사진과 수치가 생기면 CUT 01부터 CUT 12까지 순서대로 교체합니다.

이 프로젝트는 정적 HTML로 구성되어 있어 Node/npm 설치나 빌드 과정이 필요 없습니다.

## 제작 기준

- 스타일: 커피 프랜차이즈 양도양수 상세페이지형
- 현재 상태: 지역, 매출, 권리금, 임대조건 입력 전 초안
- 주의: 실제 매장 사진과 증빙자료가 없는 상태에서 제작한 판매용 구조 초안입니다.

## 실사용 전 확인 필요

- 실제 매장 외관, 내부, 장비 사진
- 매출 증빙자료와 매출 산정 기간
- 권리금, 보증금, 월세, 관리비 세부 구성
- 정확한 역명, 도보 거리, 임대조건
- 본사 가맹 승계 가능 여부와 양도 사유

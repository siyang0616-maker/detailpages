# 메가커피 양도양수 상세페이지

서울시 노원구 메가커피 양도양수 매물 상세페이지 초안입니다.

## 포함 파일

- `images/cut-01.png` ~ `images/cut-12.png`: 상세페이지 12컷 이미지
- `index.html`: 브라우저 미리보기 페이지

## 제작 기준

- 스타일: 네이버 브랜드 스토리형
- 매물 참고 정보: 서울시 노원구, 월매출 3,700만원, 창업비용 1.7억, 역세권, 기복 적은 매출, 배달 의존도 낮음
- 주의: 실제 매장 사진과 증빙자료가 없는 상태에서 제작한 판매용 초안입니다.

## 실사용 전 확인 필요

- 실제 매장 외관, 내부, 장비 사진
- 매출 증빙자료와 매출 산정 기간
- 권리금, 보증금, 월세, 관리비 세부 구성
- 정확한 역명, 도보 거리, 임대조건
- 본사 가맹 승계 가능 여부와 양도 사유

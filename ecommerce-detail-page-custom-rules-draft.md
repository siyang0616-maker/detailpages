# Ecommerce Detail Page Custom Rules Draft

## Purpose

Use these rules to customize the ecommerce-detail-page skill for franchise consulting, brand transfer, local trade-area analysis, and real-photo-based detail page production.

## Default Direction

- Default output language: Korean.
- Default sales channel: Naver Brand Story style unless the user says otherwise.
- Default cut count: 12-cut basic page.
- Default structure: problem-solving infographic plus real-photo editorial layout.
- Default target: first-time franchise buyers and buyers comparing store takeover opportunities.
- Default image style: real-photo-first, not AI-generated-looking concept art.

## Terminology Rules

- Use `창업비용` instead of `권리금`.
- Do not expose `월세` or `보증금` unless the user explicitly asks to include them.
- Prefer cautious wording such as `검토`, `확인`, `비교`, `분석`, `유리할 수 있음`.
- Do not use absolute claims such as `무조건`, `보장`, `확정`, `수익 보장`, `성공 보장`.

## Franchise Consulting Scope

When the page is for a franchise takeover, include the following analysis areas when relevant:

- 상권 분석
- 입지 진단
- 1층 여부와 노출성
- 면적과 동선
- 경쟁 브랜드 비교
- 유동인구
- 배후수요
- 창업비용 적정성
- 시설 및 장비 상태
- 인수 범위
- 본사 승인 가능성
- 매출 자료 확인
- 계약 전 체크리스트

## Real-Photo-First Image Rules

- If the user provides brand/store photos, use them as the primary visual source.
- Do not generate a fake store as the main image when real photos are available.
- Use generated visuals only to fill gaps, clean backgrounds, or create abstract supporting diagrams.
- Prefer this ratio for final detail pages:
  - 70-80% real user-provided photos
  - 20-30% design, charts, text panels, icons, and infographics
- Before production, classify photos into:
  - exterior/signage
  - interior/seating
  - counter/kiosk/order flow
  - kitchen/equipment
  - food/drink/menu
  - street/trade-area
  - promotional/menu-board materials
- Recommend which photo should be used in each cut.

## Brand Safety Rules

- Do not make the page look like official headquarters material.
- Add a small disclaimer when a franchise brand is mentioned:
  - `본 콘텐츠는 양도양수 상담/분석용이며 본사 공식 자료가 아닙니다.`
- Blur or avoid unnecessary phone numbers, faces, personal information, and unrelated store signs.
- Use brand photos as field references, but avoid implying official partnership or endorsement unless verified.

## Intake Form For Franchise Detail Pages

Collect or infer these fields:

- 브랜드명
- 지역
- 층수
- 면적
- 창업비용
- 상담 목적
- 대상 고객
- 사진 보유 여부
- 사진 유형
- 숨길 정보
- 반드시 넣을 정보
- 원하는 톤
- 컷 수
- 최종 산출물

Recommended single-question order:

1. 사진이 있는지
2. 브랜드 또는 업종
3. 상세페이지 목적
4. 대상 고객
5. 스타일
6. 컷 수
7. 필수 조건
8. 숨길 정보
9. 이미지 생성 또는 기획안 수정

## 12-Cut Default Flow For Franchise Takeover

1. Main hook
2. Buyer problem
3. Why takeover can be worth comparing
4. Store condition summary
5. Location and visibility
6. Space and operational flow
7. Facility and equipment check
8. Menu or service structure
9. New startup vs takeover comparison
10. 창업비용 review point
11. Pre-contract checklist
12. Consultation CTA

## Final QA

Before final delivery, check:

- No use of `권리금`.
- No unwanted exposure of `월세` or `보증금`.
- No fake sales numbers or profit claims.
- No official-looking brand claim.
- Korean text is readable.
- Images look grounded in real photos where photos were provided.
- Every cut has a clear purpose and CTA flow.

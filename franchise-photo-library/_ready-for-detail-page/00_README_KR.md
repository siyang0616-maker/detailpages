# 사진 정리본 사용법

이 폴더가 앞으로 상세페이지 작업 때 열어볼 최종 정리 위치입니다.

원본 사진은 건드리지 않았고, 여기에는 작업용 복사본만 브랜드별로 다시 모았습니다.

## 브랜드 순서

- `01_megacoffee`: 메가커피
- `02_subway`: 써브웨이
- `03_lotteria`: 롯데리아
- `04_composecoffee`: 컴포즈커피
- `05_woozoo-coffee`: 우지커피
- `06_twosome-place`: 투썸플레이스
- `07_baskin-robbins`: 배스킨라빈스
- `08_paris-baguette`: 파리바게뜨
- `09_paikdabang`: 빽다방

## 공통 폴더 구조

- `01_exterior_signage`: 외관, 간판, 출입구
- `02_interior_seating`: 내부, 좌석, 매장 분위기
- `03_counter_kiosk_order`: 카운터, 키오스크, 주문 동선
- `04_kitchen_equipment`: 주방, 설비, 장비 참고
- `05_food_drink_menu`: 음식, 음료, 제품, 메뉴 실물
- `06_menu_promotion`: 메뉴판, 행사물, 포스터, 홍보 이미지
- `07_street_trade_area`: 거리, 상권, 주변 환경
- `98_original_copy_reference`: 원본 복사본 전체 참고용 자리
- `99_review_needed`: 사람 노출, 흐림, 개인정보, 분류 애매함 등 최종 사용 전 확인 필요

## 상세페이지 만들 때 추천 순서

1. 첫 화면이나 신뢰 컷은 `01_exterior_signage`에서 고릅니다.
2. 매장 컨디션 설명은 `02_interior_seating`에서 고릅니다.
3. 운영 동선 설명은 `03_counter_kiosk_order`에서 고릅니다.
4. 브랜드의 상품력이나 메뉴 폭은 `05_food_drink_menu`에서 고릅니다.
5. 행사/메뉴판 근거 컷은 `06_menu_promotion`에서 고릅니다.
6. `99_review_needed`는 바로 쓰지 말고 얼굴, 번호, 민감정보를 먼저 확인하세요.

전체 현황표는 `00_overall_status.csv`에 있습니다.
